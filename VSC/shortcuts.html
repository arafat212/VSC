How to Set Shortcuts
Key Combinations:
  • Use Ctrl for Windows/Linux
  • Use Cmd for Mac
  • Use Shift for modifier
  • Use Alt or Option
  • Arrow keys: Up, Down, Left, Right
  • Function keys: F1 to F12
Examples:
  • Ctrl+Shift+Up (Increase speed)
  • Space (Play/Pause)
  • Shift+N (Next video)
  • Alt+F4 (Close player)
  • Ctrl+Alt+Shift+M (Custom shortcut)
